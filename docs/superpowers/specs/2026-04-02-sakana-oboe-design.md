# さかなおぼえモード 設計書

## 概要
琵琶湖に生息する淡水魚（外来種含む80〜100種）の写真フラッシュカード暗記モード。
既存の「とり覚え」モードと同じ画像→名前入力方式。

## 写真の条件
- Wikimedia Commonsからフリーライセンス画像を使用
- 横から全身が写っている写真を優先

## データ構造

### fish_data.js
```js
const FISH_DATA = [
  { id: "biwamasu", name: "ビワマス", category: "魚類", image: "images/fish/biwamasu.jpg", tags: ["サケ科"] },
  // ... 80〜100種
];
```

### 分類（tags）
- コイ科（ニゴロブナ、ホンモロコ、ゲンゴロウブナ、ハス、ワタカなど）
- サケ科（ビワマス）
- ハゼ科（ヨシノボリ類、ビワヨシノボリなど）
- ナマズ科（ビワコオオナマズ、ナマズ）
- ドジョウ科（アジメドジョウ、スジシマドジョウなど）
- アユ科（アユ）
- スズキ科・サンフィッシュ科・その他外来種（ブラックバス、ブルーギルなど）
- その他在来種

## 追加ファイル
| ファイル | 内容 |
|---------|------|
| `fish_data.js` | 魚データ配列 `FISH_DATA` |
| `images/fish/` | 魚の横向き全身写真 |
| `fish_credits.js` | クレジットデータ（JS） |
| `fish_credits.json` | クレジットデータ（JSON） |
| `download_fish.py` | Wikimedia Commons画像ダウンロードスクリプト |

## 変更ファイル
| ファイル | 変更内容 |
|---------|----------|
| `index.html` | モードボタン「さかなおぼえ」追加、`fish_data.js`・`fish_credits.js`のscript読み込み追加 |
| `app.js` | `currentMode === 'fish'`の分岐追加（getActiveData, getModeLabels, セッション履歴キーなど） |
| `sw.js` | キャッシュ対象にfish関連ファイル追加 |

## クイズ仕様
- とり覚えモードと完全に同じフロー
- 暗記画面 → テスト（一覧/1枚ずつ）→ 結果
- カタカナ入力で名前を回答
- 習得判定・苦手優先・ランダムモード・エクスポート/インポートすべて共通設定を使用

## サブモードなし
- しょくそう覚えのようなサブモード切替は不要
- シンプルに画像→名前の1方向のみ
